### 1.5️⃣ **הוספת שורת מידע על ריביות מעודכנות** (חובה!)
אחרי כל מופע של `[current_date format="F Y" hebrew="true"]` בדף, הוסף **מיד לאחריו** שורה חדשה:

```html
<p class="[CLASS-NAME]-rate-info" style="font-size: 0.95em; color: #666; margin-top: 10px;">
  💡 הריביות בתוכן מעודכנות לפי ריבית פריים 5.75% (ריבית בנק ישראל 4.25%, נכון לנובמבר 2025)

```

**הוראות:**
- **[CLASS-NAME]** - החלף עם שם הקלאס המתאים לעיצוב הדף (לדוגמה: `wpc-digital-loans-d7k9`)
- שמור על ה-`style` inline כדי להבטיח תצוגה נכונה
- הוסף רק אם **קיים** `[current_date...]` בדף
- הקפד להתאים לעיצוב הקיים של הדף

**דוגמה מלאה:**
```html
<div class="wpc-digital-loans-d7k9-date-update">
  <strong>עודכן לאחרונה:</strong> [current_date format="F Y" hebrew="true"]
</div>
<p class="wpc-digital-loans-d7k9-rate-info" style="font-size: 0.95em; color: #666; margin-top: 10px;">
  💡 הריביות בתוכן מעודכנות לפי ריבית פריים 5.75% (ריבית בנק ישראל 4.25%, נכון לנובמבר 2025)
```

## 🚨 כללי זהב - חובה מוחלטת!

### ❌ **אסור בהחלט:**
1. **לגעת ב-CSS** - אפילו לא רווח!
   ```html
   <!-- ❌ אסור לשנות -->
   <style>...</style>
   ```

2. **לגעת ב-JavaScript** - אפילו לא תו!
   ```html
   <!-- ❌ אסור לשנות -->
   <script>...</script>
   ```

3. **להשתמש ב-inline styles בשינויי תוכן או עיצוב:**
   ```html
   <!-- ❌ אסור להוסיף או לשנות -->
   <div style="...">...</div>
   <p style="color: red;">...</p>
   <h3 style="text-align: center;">...</h3>
   ```
   **✅ במקום זה:** השתמש רק במחלקות (classes) הקיימות של העמוד!
   ```html
   <!-- ✅ נכון - השתמש במחלקות קיימות -->
   <div class="wpc-example-title">...</div>
   <p class="wpc-rate-info">...</p>
   ```

4. **לשנות מבנה HTML** - רק תוכן!
   ```html
   <!-- ❌ אסור להוסיף/להסיר תגים -->
   <div class="...">
     <!-- ✅ רק את התוכן פה אפשר לשנות -->
   </div>
   ```

5. **לעדכן ריביות לא צמודות לפריים:**
   - ריביות קבועות (מצוין "קבועה")
   - ריביות של כרטיסי אשראי (>15%)
   - ריביות של משיכות מזומן
   - ריביות שאין להן קשר להלוואות

6. **להוסיף מסלולים לא רלוונטיים:**
   - דף על רכב → רק מסלולי רכב/ליסינג
   - דף על שיפוץ → רק מסלולי שיפוץ/משכנתא
   - אל תוסיף "הלוואה כללית" לדף ספציפי!

### 🎨 **כלל חדש קריטי - אין עוד inline styles!**

**כאשר צריך לעצב משהו:**
1. ✅ **בדוק אם יש מחלקה קיימת** שמתאימה לאלמנט
2. ✅ **השתמש במחלקה הזו** ללא שינויים
3. ❌ **לעולם אל תוסיף `style="..."`** למרכיבי תוכן
4. ❌ **אל תשנה CSS קיים** במחלקות

**דוגמאות:**
```html
<!-- ❌ אסור! -->
<h3 style="color: #1e5490; text-align: center;">כותרת</h3>
<div style="display: block; width: 100%;">תוכן</div>

<!-- ✅ נכון! -->
<h3 class="wpc-example-title">כותרת</h3>
<div class="wpc-example-calc">תוכן</div>
```

**חריג יחיד:** שורת המידע על ריביות (סעיף 1.5) - רק היא מותרת עם inline style מינימלי.

#### **ג. דוגמאות חישוב**

**דוגמה:**
```

= תשלום חודשי של 1,942 ₪"
```

**נוסחה לחישוב תשלום חודשי:**
```javascript
function calculateMonthlyPayment(principal, annualRate, months) {
    const monthlyRate = annualRate / 100 / 12;
    const payment = principal * (monthlyRate * Math.pow(1 + monthlyRate, months)) / 
                   (Math.pow(1 + monthlyRate, months) - 1);
    return Math.round(payment);
}

// דוגמה:
calculateMonthlyPayment(100000, 6.25, 60) // = 1,942 ₪
```

#### **ד. ריביות לא חוקיות (🚨 קריטי!)** ⚖️

**תקרת הריבית החוקית בישראל: 19.5%**

אם אתה רואה ריביות מעל **19.5%** בדוגמאות - זה **מכוון**! אלה דוגמאות על ריביות לא חוקיות שאנשים קיבלו.

**מה לחפש:**
```
"דני קיבל הצעה של 30,000 ש"ח ב-35% ריבית שנתית"
"חיסכון שנתי: 30,000 × (35%-19.5%) = 4,650 ש"ח"
"פיצוי כפול: 9,300 ש"ח"
```

**מה לעדכן:**
1. ✅ **את הריבית הלא חוקית** - אם השתנתה (למשל מ-35% ל-33%)
2. ✅ **את החישובים** - **חובה!** לא רק הריבית!

**נוסחאות:**
```javascript
// חיסכון שנתי מריבית לא חוקית:
function calculateIllegalRateSavings(loanAmount, illegalRate) {
    const legalCap = 19.5;
    const savings = loanAmount * (illegalRate - legalCap) / 100;
    return Math.round(savings);
}

// פיצוי כפול (לפי חוק):
function calculateDoublePenalty(savings) {
    return savings * 2;
}

// דוגמה:
const loanAmount = 30000;
const oldIllegalRate = 35;
const newIllegalRate = 33; // אם השתנתה בדוגמה

// חישוב ישן:
const oldSavings = calculateIllegalRateSavings(30000, 35);  // = 4,650 ש"ח
const oldPenalty = calculateDoublePenalty(oldSavings);       // = 9,300 ש"ח

// חישוב חדש:
const newSavings = calculateIllegalRateSavings(30000, 33);  // = 4,050 ש"ח
const newPenalty = calculateDoublePenalty(newSavings);       // = 8,100 ש"ח
```

### **5. ריביות לא חוקיות - עדכון חישובים** ⚖️

**🚨 קריטי:** אם יש דוגמה עם ריבית מעל 19.5% - **חובה לעדכן גם את החישובים!**

**דוגמה מהחיים האמיתיים:**

```html
לפני (ריבית לא חוקית 35%):
<p>דני קיבל הצעה של 30,000 ש"ח ב-35% ריבית.</p>
<p>חיסכון: 30,000 × (35%-19.5%) = 4,650 ש"ח</p>
<p>פיצוי כפול: 9,300 ש"ח</p>

אחרי (אם הריבית השתנתה ל-33%):
<p>דני קיבל הצעה של 30,000 ש"ח ב-33% ריבית.</p>
<p>חיסכון: 30,000 × (33%-19.5%) = 4,050 ש"ח</p>
<p>פיצוי כפול: 8,100 ש"ח</p>
```

**חישובים:**
- `חיסכון = סכום × (ריבית_לא_חוקית - 19.5) / 100`
- `פיצוי_כפול = חיסכון × 2`

**שים לב:**

- ❌ אל תשנה שמות (כמו "דני")
