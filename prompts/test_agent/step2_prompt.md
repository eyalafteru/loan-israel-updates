# שלב 2 - הרחבת הניתוח

## מטרה
בשלב זה תרחיב את הניתוח מהשלב הקודם ותוסיף פרטים.

## הוראות

1. **קרא את דוח שלב 1** שיצרת קודם
2. **הרחב כל בעיה** עם פירוט טכני
3. **הוסף קוד לדוגמה** לתיקונים מומלצים

## פלט נדרש

צור דוח מורחב בפורמט Markdown:

```markdown
# דוח מורחב - שלב 2

## בעיות מפורטות

### בעיה 1: [שם הבעיה]
**תיאור:** [פירוט]
**מיקום בקוד:** [שורה/אלמנט]
**תיקון מוצע:**
\`\`\`html
<!-- קוד מתוקן -->
\`\`\`

### בעיה 2: ...
```

## חשוב!
- השתמש בדוח שלב 1 כבסיס
- הוסף דוגמאות קוד קונקרטיות
