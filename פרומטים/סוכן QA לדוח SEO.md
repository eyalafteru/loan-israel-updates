### 3. 🔗 קישורים פנימיים (מקובץ הויקי!)

> **⚠️ לא לספור CTA!** קישורי WhatsApp, טלפון (tel:), וכפתורי "צור קשר" הם CTA - לא חלק מבדיקת צפיפות!

| בדיקה | יעד | סטטוס |
|-------|-----|-------|
| **צפיפות** | 1 קישור לכל 200 מילים עבריות | ✅/⚠️/❌ |
| **מיקום** | קישור ב-2 הפסקאות הראשונות | ✅/❌ |
| **עוגנים** | טבעיים ומגוונים (לא רק exact match) | ✅/❌ |
| **כפילויות** | אין קישור לאותו URL פעמיים | ✅/❌ |
| **⚠️ מקסימום לפסקה** | **לא יותר מ-2 קישורים!** | ✅/❌ |
| **⚠️ קישורים צמודים** | **לא 2 קישורים באותה שורה!** | ✅/❌ |
| **לא בכותרות** | אין קישורים ב-H1-H6 | ✅/❌ |
| **Target נכון** | פנימי=_self, חיצוני=_blank | ✅/❌ |

**קישורים שלא נספרים:**
- `wa.me/` או `api.whatsapp.com` - WhatsApp CTA
- `tel:` - קישורי טלפון
- כפתורים עם class שמכיל `btn`, `button`, `cta`
- קישורים בתוך `<footer>` או `<nav>`

**⛔ אזורים שאסור לקשר מהם בשום מקרה:**
- שורת עדכון תאריך: `עודכן לאחרונה:` / `[current_date]`
- שורת ריביות: `הריביות בתוכן מעודכנות לפי ריבית פריים` / `ריבית בנק ישראל`
- כל שורה שמכילה 💡 עם אחוזי ריבית (משתנה דינאמית!)

### 4. 🚨 בדיקת צפיפות קישורים (קריטי!)

**סרוק כל פסקה וספור קישורים:**

```
אם יש <p> עם 3+ קישורים (<a>) → ❌ בעיה!
אם יש 2 קישורים עם 1-2 מילים ביניהם → ❌ בעיה!
```

**דוגמה לבעיה:**
```html
<p>בדוק <a href="...">דירוג אשראי</a> ב<a href="...">מערכת</a>. 
   קרא על <a href="...">הלוואות</a>.</p>
<!-- ❌ 3 קישורים בפסקה אחת! -->
```
