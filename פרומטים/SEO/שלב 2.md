## 📖 קריאת קובץ חכמה - חובה!

קבצי ה-HTML במערכת מכילים CSS/JS מוטמע.
**אסור לקרוא את כל הקובץ!**

### פרוטוקול קריאה:

**שלב 1 - מצא את אזור התוכן:**
```
Grep pattern="</style>" path="[קובץ ה-HTML]"
```
רשום את מספר השורה של ה-`</style>` **האחרון** - התוכן מתחיל כ-100 שורות אחריו.

**שלב 2 - קרא רק את אזור התוכן:**
```
Read path="[קובץ]" offset=[שורה שמצאת + 100] limit=3000
```

**שלב 3 - חיפוש ממוקד לקישורים:**
- כל הקישורים: `Grep pattern="<a href" path="[קובץ]"`
- קישורים חיצוניים: `Grep pattern="target=\"_blank\"" path="[קובץ]"`
- rel attributes: `Grep pattern="rel=" path="[קובץ]"`

### ⛔ אזורים אסורים לקריאה/עריכה:
- **כל מה שלפני `</style>` האחרון** (CSS ו-JavaScript) - מצא את השורה עם Grep ואל תקרא לפניה!
- תגיות `<style>` ו-`<script>`
- namespace classes (כמו `wpc-*-d7k9-*`)

### 1. 🔗 קישורים חיצוניים (רשימת הזהב בלבד!)

**🚨 חוקים קריטיים:**

| כלל | פירוט |
|-----|-------|
| **אין להמציא קישורים** | אם הדומיין לא ברשימה - אסור לשימוש |
| **קישור יוניקי** | מקסימום קישור אחד לכל מקור חיצוני בעמוד |
| **לדף הבית בלבד** | אלא אם צוין אחרת ברשימה |
| **target="_blank"** | חובה לכל קישור חיצוני |
| **rel="noopener noreferrer"** | חובה לכל קישור חיצוני |

### 2. 🔗 קישורים פנימיים (מתוך {{INTERNAL_LINKS_DB}})

⚠️ לא לספור CTA! (וואטסאפ, טלפון, כפתורי צור קשר)

| בדיקה | יעד | סטטוס |
|-------|-----|-------|
| צפיפות | 1 קישור לכל 200 מילים עבריות | ✅/⚠️/❌ |
| מיקום | קישור ב-2 הפסקאות הראשונות | ✅/❌ |
| עוגנים | טבעיים ומגוונים (לא רק exact match) | ✅/❌ |
| כפילויות | אין קישור לאותו URL פעמיים | ✅/❌ |
| מקסימום לפסקה | לא יותר מ-2 קישורים! | ✅/❌ |
| קישורים צמודים | לא 2 קישורים באותה שורה! | ✅/❌ |
| Target נכון | פנימי=_self, חיצוני=_blank | ✅/❌ |

**✨ בדיקת "קריאות הקישור":**
- ❌ רע: "למידע נוסף על [הלוואות לעסקים לחץ כאן]."
- ✅ טוב: "במקרים מסוימים, [הלוואות לעסקים] יכולות לעזור..."
