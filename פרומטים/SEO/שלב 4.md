## 📖 קריאת קובץ חכמה - חובה!

קבצי ה-HTML במערכת מכילים CSS/JS מוטמע.
**אסור לקרוא את כל הקובץ!**

### פרוטוקול Grep-Read-Edit (חובה לפני כל עריכה!):

**שלב 1 - מצא את אזור התוכן:**
```
Grep pattern="</style>" path="[קובץ ה-HTML]"
```
רשום את מספר השורה של ה-`</style>` **האחרון** - התוכן מתחיל כ-100 שורות אחריו.

**שלב 2 - חפש קישורים:**
```
Grep pattern="<a href" path="[קובץ]"
```
רשום את מספרי השורות של הקישורים.

**שלב 3 - קרא הקשר (±10 שורות):**
```
Read path="[קובץ]" offset=[שורה-10] limit=20
```

**שלב 4 - ערוך נקודתית:**
השתמש ב-**Edit** (לא Write!) לשנות רק את השורות הספציפיות.

### חיפוש ממוקד לקישורים:
- כל הקישורים: `Grep pattern="<a href" path="[קובץ]"`
- קישורים חיצוניים: `Grep pattern="target=\"_blank\"" path="[קובץ]"`
- קישורים ללא rel: `Grep pattern="<a href.*_blank" path="[קובץ]"`
- פסקאות עם קישורים: `Grep pattern="<p.*<a href" path="[קובץ]"`

### בדיקת צפיפות קישורים:
```
Grep pattern="<p" path="[קובץ]" -A=5
```
בדוק כל פסקה - מקסימום 2 קישורים!

### ⛔ אזורים אסורים לקריאה/עריכה:
- **כל מה שלפני `</style>` האחרון** (CSS ו-JavaScript) - מצא את השורה עם Grep ואל תקרא לפניה!
- תגיות `<style>` ו-`<script>`
- namespace classes (כמו `wpc-*-d7k9-*`)

### שלב 2: תקן קישורים חיצוניים

**🚨 כלל קריטי - קישור יוניקי!**
- מקסימום קישור אחד בלבד לכל מקור חיצוני (דומיין) בעמוד!
- חובה להוסיף `target="_blank"` לכל קישור חיצוני.
- חובה להוסיף `rel="noopener noreferrer"` לכל קישור חיצוני.

### שלב 3: תקן קישורים פנימיים

השתמש בקובץ הקישורים שסופק לך:

| פעולה | מתי |
|-------|-----|
| הוסף קישורים | צפיפות נמוכה (פחות מ-1 ל-200 מילים) |
| הסר קישורים כפולים | אותו URL פעמיים בעמוד |
| תקן עוגנים | Anchor Text לא טבעי |

**דוגמאות לתיקון עוגנים:**
- ❌ לא טבעי: "למידע נוסף [לחץ כאן]."
- ✅ טבעי: "ניתן להיעזר ב[הלוואות בערבות מדינה] לצורך..."

### שלב 4: 🚨 תיקון צפיפות קישורים (קריטי!)

סרוק את ה-HTML ותקן לפי הכללים:

| בעיה | זיהוי | תיקון |
|------|-------|-------|
| עומס בפסקה | 3+ קישורים בפסקה אחת | הסר קישורים או פצל לפסקאות |
| קישורים צמודים | פחות מ-3 מילים בין קישורים | מחק אחד או הרחק ביניהם |
| קישורים בכותרת | תגית `<a>` בתוך H1-H6 | הסר את הקישור (השאר טקסט) |

**⚠️ אזורים אסורים לקישור (מחק קישורים מכאן!):**
- שורת "עודכן לאחרונה"
- שורת הריביות הדינמית (💡 הריביות בתוכן מעודכנות...)
- כל אזור עם תוכן שמשתנה אוטומטית (Shortcodes)

**אל תספור CTA:** קישורי WhatsApp, טלפון, וכפתורי "צור קשר" אינם נחשבים בצפיפות.
