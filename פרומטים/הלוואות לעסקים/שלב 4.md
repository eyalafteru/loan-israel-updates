
### 4. בדיקת תחביר HTML

| בדיקה | תקין? |
|-------|-------|
| כל התגיות נסגרות | ✅/❌ |
| אין div בתוך p | ✅/❌ |
| אין a בתוך a | ✅/❌ |
| קידוד UTF-8 | ✅/❌ |

**בעיות נפוצות לתקן:**
- תגיות `<div>` לא סגורות
- תגיות `<p>` לא סגורות
- סוגריים משולשים לא תואמים ב-Schema

---

### 4.5. בדיקת RTL (ימין לשמאל)

**⚠️ חובה: כל התוכן חייב להיות עטוף ב-div עם RTL!**

| בדיקה | תקין? |
|-------|-------|
| יש `<div dir="rtl" style="text-align: right; max-width: 1200px; ...">` פותח | ✅/❌ |
| יש `</div>` סוגר בסוף | ✅/❌ |
| ה-Schema בתוך ה-div (לפני הסגירה) | ✅/❌ |
| רוחב מקסימלי 1200px | ✅/❌ |

**מבנה נכון:**
```html
<div dir="rtl" style="text-align: right; max-width: 1200px; margin: 0 auto; padding: 0 20px;">
  <!-- כל התוכן כולל H1, פסקאות, FAQ -->
  <!-- Schema JSON-LD -->
</div>
```
