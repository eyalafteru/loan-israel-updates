## ⚠️ חשוב מאוד: עריכה על הקובץ הקיים!

**אל תיצור קובץ חדש!** השתמש בכלי **Edit** לערוך את קובץ ה-HTML המקורי ישירות.

**הדרך הנכונה:**
1. קרא את הקובץ המקורי עם Read
2. השתמש ב-**Edit** לעשות שינויים נקודתיים (לא Write!)
3. כל תיקון = קריאת Edit נפרדת

**למה Edit ולא Write?**
- Edit משנה רק את החלק הרלוונטי
- Write דורס את כל הקובץ (מסוכן!)
- Edit שומר על כל העיצוב והמבנה

📁 **חובה לשמור דוח!** הנתיב המדויק יינתן לך בפרומפט של המערכת.

## 2. סוגי תיקונים ואיך ליישם

### א. תיקון H1 (רק אם קריטי!)

**הדוח יציין במפורש אחד מאלה:**
- "H1 כפול - יש למחוק/לשנות"
- "H1 חסר לגמרי"
- "H1 ספאמי - דורש שינוי"

**אם לא צוין - אל תיגע ב-H1!**

```html
<!-- אם הדוח מבקש לתקן H1 ספאמי -->
<!-- לפני -->
<h1>הלוואה הלוואה הלוואה הכי טוב הלוואה</h1>

<!-- אחרי -->
<h1>הלוואה לעסקים - תנאים מותאמים אישית</h1>
```

### ב. תיקון H2-H6 Keyword Stuffing

**אם הדוח מציין כותרות עם ספאם מילות מפתח:**

```html
<!-- לפני - ספאמי -->
<h2>הלוואה הלוואה לעסקים הלוואה מהירה</h2>

<!-- אחרי - מגוון -->
<h2>מימון מהיר לעסקים - תנאים מותאמים</h2>
```

### ג. הסרת Bold Spam

**מצא והסר הדגשות מיותרות על מילות מפתח:**

```html
<!-- לפני - ספאמי -->
<p>אם אתם מחפשים <strong>הלוואה</strong> מהירה, <strong>הלוואה</strong> זמינה 
   דרך <strong>הלוואה</strong> חוץ בנקאית.</p>

<!-- אחרי - נקי -->
<p>אם אתם מחפשים מימון מהיר, <strong>הלוואה חוץ בנקאית</strong> זמינה 
   במספר ערוצים מורשים.</p>
```

### ד. תיקון Schema JSON-LD

**אם הדוח מציין סכמה חסרה/שגויה:**

**⚠️ פרטי Schema נכונים - חובה!**

| שדה | ערך נכון | ❌ לא לכתוב |
|-----|----------|-------------|
| **Person.name** | `אייל עובדיה` | ❌ "מומחה הלוואות" |
| **Person.jobTitle** | `המנהל המקצועי והיועץ הראשי` | ❌ "יועץ פיננסי", ❌ "מנכ"ל" |
| **Organization.name** | `רק תבקש` (פורטל) או `אפטריו בע"מ` (חברה) | ❌ "הלוואות ישראל" |
| **Organization.url** | `https://loan-israel.co.il` | |

**פרטי החברה המפעילה:**
- **שם החברה:** אפטריו בע"מ
- **ח.פ:** 514207372

**דוגמה נכונה:**
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "author": {
    "@type": "Person",
    "name": "אייל עובדיה",
    "jobTitle": "המנהל המקצועי והיועץ הראשי"
  },
  "publisher": {
    "@type": "Organization",
    "name": "רק תבקש",
    "url": "https://loan-israel.co.il",
    "parentOrganization": {
      "@type": "Corporation",
      "name": "אפטריו בע\"מ",
      "taxID": "514207372"
    }
  },
  "mainEntity": [...]
}
</script>
```

## 6. דוגמה מלאה

### דוח שלב 1 אומר:

```
Keyword Stuffing בכותרות:
- H2: "הלוואה הלוואה מהירה הלוואה לעסקים" → לשנות ל: "מימון מהיר לעסקים"

Bold Spam:
- שורה 145: "<strong>הלוואה</strong> זמינה <strong>הלוואה</strong> מהירה" → להסיר הדגשה כפולה
```

```html

```

```html
<h2>מימון מהיר לעסקים - תנאים מותאמים</h2>
...
<p>אם אתם מחפשים מימון זמין, <strong>הלוואה מהירה</strong> היא הפתרון.</p>
```
