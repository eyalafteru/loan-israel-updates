## 2. סוגי תיקונים ואיך ליישם

### א. תיקון/הוספת דיסקליימר 📋

**אם הדיסקליימר חסר לגמרי:**

מצא את סוף ה-`<footer>` או האזור האחרון לפני `</body>` והוסף:

```html
<p style="font-size: 0.8em; color: #666; text-align: center; padding: 20px; direction: rtl;">
    [הטקסט המדויק מהדוח]
</p>
```

**אם הדיסקליימר קיים אבל שגוי:**

מצא את הפסקה הקיימת והחלף רק את הטקסט הפנימי.

### ב. תיקון ביטויים בעייתיים (YMYL) ⚠️

**הדוח אומר:**
> במקום: "אישור מיידי לכולם" -> שנה ל: "בדיקת זכאות מהירה"

**הפעולה:**
1. חפש את הביטוי המדויק ב-HTML
2. החלף רק את הטקסט, שמור על כל ה-tags וה-styles
3. אל תשנה שום דבר סביב הביטוי

```html
<!-- לפני -->
<p class="wpc-promo-text">אישור מיידי לכולם!</p>

<!-- אחרי -->
<p class="wpc-promo-text">בדיקת זכאות מהירה</p>
```

### ג. תיקון H1 (רק אם קריטי!)

**הדוח יציין במפורש אחד מאלה:**
- "H1 כפול - יש למחוק/לשנות"
- "H1 חסר לגמרי"
- "H1 ספאמי - דורש שינוי"

**אם לא צוין - אל תיגע ב-H1!**

```html
<!-- אם הדוח מבקש לתקן H1 ספאמי -->
<!-- לפני -->
<h1>הלוואה הלוואה הלוואה הכי טוב הלוואה</h1>

<!-- אחרי -->
<h1>הלוואה לעסקים - תנאים מותאמים אישית</h1>
```

### ד. העשרת תוכן (Trust Signals)

**הדוח יציע משפט כמו:**
> "הוסף: רק תבקש פועלת מאז 2015 ומחוברת ליותר מ-30 גופים פיננסיים מורשים."

**מצא מיקום מתאים:**
- אזור "אודות" או "למה אנחנו"
- סוף פסקת פתיחה
- לפני ה-CTA הראשי

**הוסף בסגנון הקיים:**
```html
<p class="[same-class-as-nearby]" style="[copy-existing-style]">
    רק תבקש פועלת מאז 2015 ומחוברת ליותר מ-30 גופים פיננסיים מורשים.
</p>
```

## 6. דוגמה מלאה

### דוח SEO אומר:

```
תיקון דיסקליימר:
החלף את הקיים בטקסט הבא:
"בכפוף לתקנון הקרן המנהלת. אי עמידה בפירעון עלולה לגרור חיוב ריבית פיגורים"

במקום: "אישור מובטח" -> שנה ל: "בדיקת זכאות מהירה"

```

### לפני:
```html
<p class="wpc-promo">אישור מובטח תוך דקות!</p>
...

    <p>תנאים והגבלות חלים</p>
</footer>
```

### אחרי:
```html
<p class="wpc-promo">בדיקת זכאות מהירה תוך דקות!</p>

<footer>
    <p>בכפוף לתקנון הקרן המנהלת. אי עמידה בפירעון עלולה לגרור חיוב ריבית פיגורים</p>
```
