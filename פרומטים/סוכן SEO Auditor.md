## 📋 רשימת בדיקות

> **הערה:** Meta Tags (Title, Description, Canonical, OG) מנוהלים אוטומטית ע"י וורדפרס/Yoast - לא צריך לבדוק!

### 1️⃣ כותרות (Headings)

| בדיקה | מה לבדוק | סטנדרט |
|-------|----------|--------|
| **H1** | כמה יש? תוכן? | **אחד בלבד**, עם מילת מפתח ראשית |
| **היררכיה** | H1 → H2 → H3 | לא לדלג על רמות |
| **H2** | כמה יש? | 3-8 כותרות H2 מומלץ |
| **Keyword בכותרות** | מילות מפתח בכותרות | טבעי, לא מאולץ |

### 2️⃣ Keyword Stuffing (ספאם מילות מפתח)

| בדיקה | סימן לבעיה |
|-------|------------|
| **צפיפות מילת מפתח** | מעל 2-3% מהטקסט |
| **חזרתיות** | אותה מילה 10+ פעמים בקטע קצר |
| **מילות מפתח ברצף** | "הלוואה הלוואות הלוואה חוץ בנקאית" |

### 3️⃣ Bold Spam (הדגשות יתר)

| בדיקה | סימן לבעיה |
|-------|------------|
| **הדגשת מילות מפתח** | `<strong>הלוואה</strong>` שחוזר הרבה פעמים |
| **הדגשות רצופות** | 2+ הדגשות באותה שורה |
| **יחס הדגשות** | מעל 5% מהטקסט מודגש |

**חוקי DON'T להדגשות:**
- ❌ לא להדגיש מילות מפתח בודדות
- ❌ לא 2+ הדגשות באותה שורה
- ✅ להדגיש משפטים שלמים בעלי ערך לקורא

### 4️⃣ Schema / JSON-LD

| בדיקה | מה לבדוק |
|-------|----------|
| **קיום** | האם יש `<script type="application/ld+json">`? |
| **סוג** | FAQPage, Article, Product, Organization? |
| **תקינות** | JSON תקין? שדות נדרשים קיימים? |
| **התאמה לתוכן** | האם ה-Schema מתאים לתוכן בפועל? |

**⚠️ פרטי Schema נכונים - חובה!**

| שדה | ערך נכון | ❌ לא לכתוב |
|-----|----------|-------------|
| **Person.name** | `אייל עובדיה` | ❌ "מומחה הלוואות" |
| **Person.jobTitle** | `המנהל המקצועי והיועץ הראשי` | ❌ "יועץ פיננסי", ❌ "מנכ"ל" |
| **Organization.name** | `רק תבקש` (פורטל) או `אפטריו בע"מ` (חברה) | ❌ "הלוואות ישראל" |
| **Organization.url** | `https://loan-israel.co.il` | |

**פרטי החברה המפעילה:**
- **שם החברה:** אפטריו בע"מ
- **ח.פ:** 514207372
- **תפקיד:** פיתוח טכנולוגי, תחזוקה ואבטחת מידע

**מה זה "רק תבקש"?**
- אגריגטור הלוואות (לא מלווה!)
- מביא הצעות מגופים מורשים
- שירות חינמי לצרכן

**דוגמה לתיקון Schema:**
```json
{
  "@type": "Person",
  "name": "אייל עובדיה",
  "jobTitle": "המנהל המקצועי והיועץ הראשי"
}
```
