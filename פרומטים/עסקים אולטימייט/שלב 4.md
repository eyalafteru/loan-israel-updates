### 3. בדיקת Schema Markup ⭐

**חפש בעמוד:**
```
grep -n "application/ld+json" [קובץ]
```

| Schema | קיים? | תקין? | הערות |
|--------|-------|-------|-------|
| Person (מומחה) | ✅/❌ | ✅/❌ | |
| Organization | ✅/❌ | ✅/❌ | |
| BreadcrumbList | ✅/❌ | ✅/❌ | |
| FAQPage | ✅/❌ | ✅/❌ | חייב להתאים ל-FAQ בעמוד! |
| FinancialProduct (אם רלוונטי) | ✅/❌ | ✅/❌ | |

**בדיקות Schema חובה:**
- [ ] כל ה-JSON תקין (סוגריים סגורים נכון)
- [ ] אין פסיקים מיותרים בסוף arrays/objects
- [ ] URLs מלאים עם `https://`
- [ ] `@context` הוא `https://schema.org`
- [ ] תאריכים בפורמט ISO (`2026-01-12`)
- [ ] FAQPage מכיל את אותן שאלות שבעמוד

### 4. בדיקת תחביר HTML

| בדיקה | תקין? |
|-------|-------|
| כל התגיות נסגרות | ✅/❌ |

| בעיה | פתרון |
|------|-------|

| `<p>` לא סגור | הוסף `</p>` |
| סוגריים לא תואמים ב-Schema | בדוק `{}` ו-`[]` |

### 5. בדיקת RTL (ימין לשמאל) ⭐

**⚠️ חובה: כל התוכן חייב להיות מכוון RTL!**

| בדיקה | תקין? |
|-------|-------|
| יש `<div dir="rtl" ...>` פותח | ✅/❌ |
| יש `</div>` סוגר בסוף | ✅/❌ |
| `text-align: right` | ✅/❌ |
| `max-width: 1200px` | ✅/❌ |
| `margin: 0 auto` | ✅/❌ |
| Schema בתוך ה-div | ✅/❌ |

**מבנה נכון:**
```html
<div dir="rtl" style="text-align: right; max-width: 1200px; margin: 0 auto; padding: 0 20px;">
  <!-- H1, תוכן, FAQ -->
  <!-- Schema JSON-LD -->
</div>
```

### 6. בדיקת YMYL (Your Money Your Life) ⭐

| בדיקה | תקין? |
|-------|-------|
| אין "100% אישור" / "מובטח" | ✅/❌ |
| אין "הכי זול בישראל" | ✅/❌ |
| יש הסתייגויות ("בכפוף ל...", "תלוי ב...") | ✅/❌ |
| מקורות למידע פיננסי (בנק ישראל) | ✅/❌ |
| דיסקליימר קיים | ✅/❌ |
| הנתונים מעודכנים | ✅/❌ |

**ביטויים אסורים:**
- ❌ "אישור מובטח"
- ❌ "100% הצלחה"
- ❌ "הריבית הכי נמוכה"
- ❌ "ללא בדיקות"

**ביטויים מומלצים:**
- ✅ "בהתאם לפרופיל הלקוח"
- ✅ "בכפוף לאישור"
- ✅ "עד X%" (לא "X% מובטח")
