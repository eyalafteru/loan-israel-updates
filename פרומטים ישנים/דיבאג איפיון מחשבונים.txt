# 🔍 פרומפט דיבאג איפיון מחשבונים - בדיקת איכות מקיפה v1.0

## 🚨 הוראות שימוש
**העתק את האיפיון שיצא בקנבס ובקש:** "בדוק את האיפיון הזה לפי כל הקריטריונים"

**🎯 מה הדיבאג בודק:**
- איפיון מלא אחד שיצא בקנבס
- האיפיון כולל הוראות לקורסר לעבוד בשני שלבים
- בדיקה שההוראות לקורסר ברורות ומפורטות

---

## 🎯 משימת הדיבאג
אתה בודק איפיון מחשבון שיצא מהמחולל כנגד כל הדרישות המקוריות, **מתקן את כל הבעיות ישירות בקנבס**, ומוציא דוח קצר של מה שתוקן.

**🚨 חשוב: אל תרשום מה צריך לתקן - פשוט תקן הכל בקנבס ותן דוח קצר בסוף!**

### 🎨 1. בדיקת מפרט טכני לקורסר

**🔍 חובה להופיע בתחילת האיפיון:**

- [ ] רכיבים ספציפיים לכל טאב (לא גנריים)
- [ ] אזור AWG עם כפתור CTA (מוסתר עם opacity:0 ו-height:0 - לא display:none!)
- [ ] חיפושים ספציפיים בvisual.txt (לא רק הבסיסיים)
- [ ] נוסחאות מתמטיות ספציפיות למחשבון
- [ ] JavaScript namespace דינמי נכון
- [ ] **רוחב 1200px מוגדר:** max-width: 1200px לכל wrapper/container
- [ ] **AWG רוחב מלא ורקע לבן:** [awg postid="32400"] עם width: 100% ו-background: white

**❌ בעיות נפוצות לחפש:**
- מפרט טכני חסר או בסוף האיפיון
- טאבים גנריים כמו "בסיסי/מתקדם/השוואה"

- נוסחת PMT גנרית לכל מחשבון

### 📱 20. בדיקת תיקון צבעים למכשירים אמיתיים - קריטי!

**🔍 בדיקת CSS למכשירים אמיתיים:**
- [ ] **color-scheme: light בראש הקובץ:** כפיית מצב בהיר על כל המכשירים
- [ ] **forced-color-adjust: none:** מניעת שינוי צבעים אוטומטי
- [ ] **@media (prefers-color-scheme: dark):** כפיית בהיר גם במצב כהה
- [ ] **-webkit-text-fill-color: initial:** ביטול שינויי WebKit

**🔍 בדיקת fallback colors במובייל:**
- [ ] **@media (max-width: 768px):** fallback colors לכל האלמנטים הבעייתיים
- [ ] **@media (max-width: 480px):** חזרה על אותם תיקונים
- [ ] **@media (max-width: 375px):** תיקונים גם לרזולוציה קטנה
- [ ] **אלמנטים כלולים:** tabs, labels, headers, comparison-cards, case-params, faq-headers

**❌ בעיות נפוצות במכשירים אמיתיים:**
- טקסט לבן במקום שחור (CSS Variables לא נטענים)
- הבדלים בין דפדפן למכשיר אמיתי
- מצב כהה משפיע על צבעים
- WebKit משנה צבעי טקסט

**✅ הפתרון הנכון:**
```css

    -webkit-forced-color-adjust: none !important;
    -ms-high-contrast-adjust: none !important;
}

@media (prefers-color-scheme: dark) {
    .wpc-calc-[topic]-[4random]-wrapper {
        color-scheme: light !important;
        background-color: white !important;
    }
}

/* fallback colors במובייל */
@media (max-width: 768px) {
    .wpc-calc-[topic]-[4random]-tab:not(.active) { color: #222222 !important; }
    .wpc-calc-[topic]-[4random]-example-label { color: #222222 !important; }
    /* וכו' לכל האלמנטים הבעייתיים */
}
```
