## 🔧 קוד JavaScript

### מבנה:
```javascript
(function() {
  'use strict';
  const ns = 'WPC_Loans_D7M3';
  if (window[ns]) return; // מניעת כפילויות
  window[ns] = true;

  const container = document.querySelector('.wpc-loans-d7m3-faq-container');
  if (!container) return;

  // Event Delegation על container בלבד
  container.addEventListener('click', function(e) {
    const button = e.target.closest('[data-action="toggle-faq"]');
    if (!button) return;
    // Toggle logic...
  });
})();
```

### תכונות:
- ✅ IIFE לבידוד Scope
