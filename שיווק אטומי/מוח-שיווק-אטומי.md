# 🧠 מוח שיווק אטומי - מדריך לכתיבת תוכן

> קובץ זה מכיל את כל העקרונות, המונחים והתבניות לכתיבת תוכן בשיטת השיווק האטומי.
> השתמש בו כ-context/instructions עבור AI.

---

## 1. עקרונות יסוד

### העיקרון המרכזי: דיבור לשני החלקים
כל אדם מורכב משני חלקים מנוגדים. תוכן אפקטיבי חייב לדבר לשניהם:

| העיגול ⭕ | המשולש 🔺 |
|-----------|-----------|
| רוצה להצליח | מפחד מהמחיר |
| אופטימי | ספקן |
| מוכן לשינוי | רוצה להישאר אותו דבר |
| "הילד הנבחר" | "הילד המורד" |

**כלל הזהב:** אם דיברת רק לעיגול - המשולש יחסום. אם דיברת רק למשולש - העיגול לא יתלהב.

### 6 עקרונות נוספים:

1. **מטרה עם שריטה** - הלקוח רוצה להגיע למטרה בלי לוותר על מה שהוא (השריטה שלו)
2. **A2-B2 לפני מכירה** - קודם משנים אמונות, אחר כך מוכרים
3. **תקרת זכוכית** - להסביר למה לא הצליח עד עכשיו (לא באשמתו!)
4. **רגש מטרה** - לסיים בהתרגשות אמיתית, לא מניפולציה
5. **הצעה שפרייר לא קונה** - מחיר שמרגיש "טיפש לא לקנות"
6. **MVP בינוני = מצוין** - לא לחכות למושלם

## 2. מילון מונחים

### מודל A1-A2-B1-B2

| מונח | הגדרה | דוגמה (דיאטה) |
|------|--------|---------------|
| **A1 - השקר** | מה אמרו לו שלא נכון | "תאכל פחות, תזוז יותר" |
| **A2 - הגילוי** | האמת שגילינו על הבעיה | "הבעיה היא הורמונלית, לא קלורית" |
| **B1 - האשליה** | איך חשב להגיע למטרה | "דיאטות קשות + ספורט אינטנסיבי" |
| **B2 - הגאולה** | הדרך החדשה שמשלבת את השריטה | "לאכול לשובע, בלי ספורט, בלי לספור" |

### מונחים מרכזיים

| מונח | הגדרה |
|------|--------|
| **לקוח LTV** | הלקוח שמקסימום מתאים לנו - יש לו את החלום, הפער, הכאבים והמוכנות |
| **שריטה** | חסרון שהוא בעצם מתנה - מה שעושה אותנו ייחודיים |
| **מפת פיצוח** | רצף האמונות שצריך לשנות כדי להגיע מ-A1 ל-B2 |
| **משולש תוכן אטומי** | מבנה לשינוי אמונה: טענה → הוכחה → תועלת |
| **הצעה אטומית** | הצעה שמדברת לשני החלקים ומרגישה "טיפש לא לקנות" |
| **מעמד מכירה** | שלב השפעה (שינוי אמונות) + שלב המרה (הצעה) |
| **בורות וסולמות** | מקומות שלקוחות נתקעים (בורות) + פתרונות (סולמות) |

## 3. מבנים ותבניות

### 3.1 משולש תוכן אטומי (CPBG)

השתמש במבנה זה לכל תוכן שמטרתו לשנות אמונה:

```
1. טענה (Claim)
   האמונה החדשה שאתה רוצה שיאמין

2. הוכחה (Proof) - בחר אחד או יותר:
   - קייס (סיפור של לקוח/אדם)
   - מחקר/נתון
   - היגיון/אנלוגיה
   - ניסוי אישי

3. תועלת (Benefit)
   מה יוצא לו אם יאמין באמונה החדשה

4. תקרת זכוכית (Glass Ceiling)
   למה לא הצליח עד עכשיו - בלי אשמה!
```

### 3.2 מבנה מודעה/פוסט

```
שורה 1: [שיקוף הקונפליקט - מדבר לשני החלקים]
שורה 2-3: [השלכה רגשית של A1 - מה זה עולה לו]
שורה 4: [רמז ל-A2 - יש דרך אחרת]
שורה 5: [קריאה לפעולה עם תועלת ספציפית]
```

### 3.3 מבנה דף המרה (הרשמה)

```
כותרת: [מדברת לשני החלקים - גם למטרה וגם לשריטה]
תת-כותרת: [מה ילמד/יקבל]

בולטים (3-5):
• [תועלת 1 - לעיגול]
• [תועלת 2 - למשולש]
• [תועלת 3 - לעיגול]

מי אני: [הצגה עצמית עם השריטה שלי - לא פרפקט!]

למה עכשיו: [דחיפות אמיתית, לא מניפולטיבית]

כפתור: [פעולה + מה קורה מיד אחרי]
```

### 3.4 מבנה מעמד מכירה

```
שלב השפעה (75%):
├── פתיחה: יצירת אמון וסקרנות
├── A1→A2: שינוי הבנת הבעיה
├── B1→B2: שינוי הבנת הפתרון
└── הפריצה הגדולה: רגע ה"אהה!"

שלב המרה (25%):
├── מטרת התהליך: מה יקבל
├── פיצ'רים מרכזיים: 3-4 בלבד
├── תמיכה וקהילה: איך נלווה אותו
├── מחיר: ברמת "פרייר לא קונה"
├── שאלות נפוצות: כל תשובה חוזרת ל-A2-B2
└── סגירה: רגש מטרה אמיתי
```

### 3.5 מבנה שיחת מכירה קלה (15-18 דקות)

```
1. הצטרפות (90 שניות): בניית אמון
2. קליטה (3-4 דקות): שאלות להבנת A1-B1
3. לימוד (7-8 דקות):
   - לימוד מנגנון: העברה מ-A1 ל-A2
   - לימוד הצעה: חיבור פיצ'רים לצרכים
4. המרה (2-3 דקות): קריאה לפעולה ברגש מטרה
```

## 4. שאלות מנחות לכתיבה

### לפני כתיבת כל תוכן, שאל:

**על הקהל:**
- מה העיגול שלו רוצה? (המטרה)
- ממה המשולש שלו מפחד? (ההתנגדות)
- מה השריטה שלו? (מה הוא לא מוכן לוותר עליו)

**על המסר:**
- מה ה-A1 שצריך לשבור? (האמונה הישנה)
- מה ה-A2 שצריך לבנות? (האמונה החדשה)
- איזה קייס/הוכחה הכי חזקה?
- מה התקרת הזכוכית? (למה לא הצליח עד עכשיו)

**על הסגירה:**
- מה הפעולה הספציפית שאני רוצה?
- איזו תועלת מיידית הוא מקבל?
- האם דיברתי לשני החלקים?

## 5. דוגמאות

### דוגמה 1: מודעה (דיאטה)

```
❌ גרסה רגילה:
"רוצה לרזות? הצטרף לתוכנית שלנו!"

✅ גרסה אטומית:
"נמאס לך לרוץ על הליכון בלי לראות תוצאות?
אני מבין. גם אני חשבתי שצריך לסבול כדי לרזות.
עד שגיליתי שהבעיה היא לא כמה קלוריות שורפים - אלא מה חוסם את ההורמונים.
לחץ כאן לגלות למה דיאטות לא עובדות (ומה כן)"
```

**ניתוח:**
- שורה 1: שיקוף קונפליקט (עיגול רוצה תוצאות, משולש לא רוצה לסבול)
- שורה 2: הזדהות עם השריטה
- שורה 3: רמז ל-A2 + תקרת זכוכית
- שורה 4: קריאה לפעולה עם סקרנות

### דוגמה 2: כותרת דף המרה

```
❌ גרסה רגילה:
"וובינר חינמי: איך לרזות ב-30 יום"

✅ גרסה אטומית:
"איך לרדת 8 קילו בלי דיאטות, בלי ספורט, ובלי להגיד לעצמך 'לא'"
```

**ניתוח:**
- "לרדת 8 קילו" = מטרה ברורה (עיגול)
- "בלי דיאטות, בלי ספורט" = מדבר למשולש
- "בלי להגיד לעצמך לא" = השריטה (לא לוותר על הנאות)

### דוגמה 3: תקרת זכוכית

```
❌ גרסה מאשימה:
"הבעיה שלך היא שאתה לא מספיק ממושמע"

✅ גרסה אטומית:
"הסיבה שזה לא עבד עד עכשיו היא לא שאתה חסר כוח רצון.
הבעיה היא שלימדו אותך שיטה שנוגדת את הביולוגיה שלך.
כשתלמד לעבוד עם הגוף ולא נגדו - הכל ישתנה."
```

## 6. טעויות נפוצות להימנע

| ❌ לא לעשות | ✅ במקום זה |
|-------------|-------------|
| לדבר רק למטרה | לשלב גם את השריטה |
| להאשים את הלקוח | להסביר את תקרת הזכוכית |
| למכור לפני A2-B2 | קודם לשנות אמונות |
| דחיפות מזויפת | דחיפות אמיתית או בלי |
| הצעה עמוסה בפיצ'רים | 3-4 פיצ'רים מרכזיים |
| "אני הכי טוב" | "זו השריטה שלי" |
| לחכות למושלם | להוציא MVP ולשפר |

## 7. צ'קליסט לפני פרסום

- [ ] האם דיברתי לעיגול? (מטרה, תקווה, התרגשות)
- [ ] האם דיברתי למשולש? (פחדים, התנגדויות, שריטה)
- [ ] האם יש A2 ברור? (למה הבעיה לא מה שחשב)
- [ ] האם יש B2 ברור? (דרך חדשה שמכבדת את השריטה)
- [ ] האם יש תקרת זכוכית? (למה לא הצליח - בלי אשמה)
- [ ] האם יש קריאה לפעולה ספציפית?
- [ ] האם הסגירה ברגש מטרה אמיתי?

## 8. פרומפט מוכן לשימוש

כשאתה רוצה ש-AI יכתוב תוכן, העתק את זה:

```
אתה מומחה לשיווק אטומי. כתוב [סוג התוכן] על [הנושא].

קהל היעד:
- מטרה: [מה רוצה להשיג]
- שריטה: [מה לא מוכן לוותר עליו]
- A1: [מה חושב שהבעיה]
- A2: [מה הבעיה באמת]

הנחיות:
1. דבר לשני החלקים (עיגול ומשולש)
2. השתמש במבנה משולש תוכן: טענה → הוכחה → תועלת
3. הוסף תקרת זכוכית (למה לא הצליח - בלי אשמה)
4. סיים בקריאה לפעולה ברגש מטרה
```
